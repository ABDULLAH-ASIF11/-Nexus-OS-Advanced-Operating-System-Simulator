# -Nexus-OS-Advanced-Operating-System-Simulator
Nexus OS is a C++ Operating System Simulator built with POSIX system calls. It features a Multilevel Queue Scheduler, real-time resource management (RAM/HDD/Cores), and IPC via pipes. Utilizing fork() and exec(), it manages 20+ concurrent tasks in independent xterm windows, using Mutex and Semaphores for process synchronization and safety.
